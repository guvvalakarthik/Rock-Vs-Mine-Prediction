# Rock vs Mine Prediction using Machine Learning

## Project Overview
This project uses Machine Learning techniques to classify sonar signals
as either **Rock** or **Mine** based on their frequency energy patterns.

The system is useful in underwater object detection and naval applications.

---

## Dataset Description
- Sonar Dataset
- 208 samples
- 60 numerical features (sonar frequency bands)
- Target classes:
  - R → Rock
  - M → Mine

---

## Model Used
- Logistic Regression

---

## Performance
- Training Accuracy: ~83%
- Testing Accuracy: ~76%

---

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn

---

## How to Run the Project
1. Clone the repository
2. Open the notebook from `notebooks/`
3. Run all cells
4. Modify `input_data` to test predictions
